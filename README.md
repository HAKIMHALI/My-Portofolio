## ⚙️ How I Work (Real Case Approach)

> Dalam setiap project, gue gak langsung "bikin visual". Gue jalanin proses yang jelas:

### 🔄 Langkah-Langkah yang Sudah Gue Jalani

| Step | Action | Fokus |
|---|---|---|
| **1** | **Breakdown Objective** | Ini buat branding atau conversion? awareness atau selling? |
| **2** | **Define Hook & Message** | Apa yang bikin orang berhenti scroll dalam 1–2 detik |
| **3** | **Build Visual Concept** | Direction, mood, composition, storytelling |
| **4** | **Execution** *(AI + Editing + Motion)* | Produksi visual dengan kontrol penuh *(no random result)* |
| **5** | **Refinement** | Fix detail kecil: lighting, texture, timing, realism |
| **6** | **Optimization for Platform** | Disesuaikan buat IG / TikTok *(biar perform, bukan cuma bagus)* |

---

### 🛠️ Masalah yang Sering Gue Hadapi (dan Cara Gue Solve)

| ❌ Problem | ✅ Solution |
|---|---|
| Visual bagus tapi gak "jual" | Balik ke strategy — fix hook & message dulu sebelum visual |
| AI output keliatan fake / aneh | Kontrol dari prompt + layering + manual correction *(bukan one-click generate)* |
| Motion keliatan random / gak natural | Treat kayak directing video — bukan sekadar animasi |
| Brand keliatan biasa aja *(gak premium feel)* | Upgrade lewat composition, lighting, dan storytelling *(bukan cuma edit warna)* |
| Content banyak tapi gak perform | Fokus ke impact per content, bukan quantity |
